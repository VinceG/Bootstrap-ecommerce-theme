Bootstrap E-Commerce Theme
=========================

This is a simple Bootstrap 2.x theme used for an eCommerce website. Check the demo link, or screenshot below.

## Pages

1. Home
2. Checkout
3. Order Placement
4. Order Status
5. About Us
6. Contact Us
7. Categories List
8. Product Page
